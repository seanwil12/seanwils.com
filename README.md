## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

Currently, there is a template site on the site and I would like to replace it with something completely custom and visually appealing. This includes fully responsive design, high resolution images, access to all apps and an admin section. There should be no cost to this goal outside of my own personal time.

Project Goals:
* Modern and responsive design.
* Should demonstrate my skill set as a tech enthusiast.
* High resolution images.
* Easy access to personal applications.
* Ability to log in and out to administer different portions of the site.
* Interactive portions to be written in Node.js and React.js

## Getting Started

Follow these steps if you are interested in cloning the project and using it for your projects. 

### Prerequisites

First, you would want to install node package manager. 
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the local dev server
   ```sh
   npm run dev
   ```
